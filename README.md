# Plates With P's Chicago Restaurant Week Guide 2026

A guide created in collaboration with Chicago content creator Paige Serena to find the best deals during [Chicago Restaurant Week 2026](https://www.choosechicago.com/chicago-restaurant-week/).

## About

This project provides an interactive guide that analyzes Chicago Restaurant Week participating restaurants, comparing their special menus against regular pricing to help diners identify the best values. The guide features a clean, sophisticated design with an elegant color palette inspired by fine dining aesthetics.

## Features

- **Restaurant Comparison Tool**: View detailed analysis of Restaurant Week menus vs. regular pricing
- **Value Calculator**: Estimated savings and value ratings for each participating restaurant
- **Responsive Design**: Optimized for both desktop and mobile viewing
- **Elegant UI**: Custom design with Cormorant Garamond and DM Sans fonts
- **Comprehensive Disclaimer**: Transparent methodology and calculation notes

## Design

The site features a warm, sophisticated design with:
- Color palette: Cream, terracotta, sage, and charcoal tones
- Typography: Cormorant Garamond (serif) for headings, DM Sans for body text
- Responsive layout optimized for all screen sizes

## Pages

- **index.html**: Main guide with restaurant listings and comparisons
- **disclaimer.html**: Methodology, disclaimers, and support information

## Local Development
